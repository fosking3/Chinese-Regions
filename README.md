# A selection panel of Chinese administrative regions
# 中国行政区划选择面板

## Usage(2 steps):

1. Import the js file, like:<br>
```
<script type="text/javascript" src="chinese-regions-1.0.0.js"></script>
```
2. Define a div, like:
```
<div class="regions"></div>
```
